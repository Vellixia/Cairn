## Steps

### Step 1: `cairn documents ingest README.md`
**Do**: run the ingest subcommand against the repo's real `README.md` (mounted into
`cairn-rustbuild` at `/work/README.md`).
**Request**: CLI reads the file locally, then `POST /api/documents/ingest` with the fetched
content.
**Expected**: prints `Ingested: README.md (N chunks)` with N > 0.
**Observed**: `Ingested: README.md (13 chunks)`
**Result**: PASS

### Step 2: `GET /api/documents` lists it
**Request**:
```http
GET /api/documents HTTP/1.1
Cookie: cairn_session=...
```
**Expected**: one entry, `source: "README.md"`, `chunk_count: 13`.
**Observed**: `[{"id":"b33563055168","source":"README.md","title":"README.md","chunk_count":13,"updated_at":"2026-07-03T04:25:18.317Z"}]`
**Result**: PASS

### Step 3: `cairn documents search` finds relevant chunks
**Do**: `cairn documents search "memory tiers"`.
**Expected**: at least one `[README.md]`-tagged chunk with real content.
**Observed**: returned a `[README.md]` chunk about the pack/registry/federation feature list -
a real, present-in-the-file passage.
**Result**: PASS

### Step 4: `GET /api/context/assemble` surfaces `[doc]`-sourced chunks alongside memory results
**Do**: `GET /api/context/assemble?q=cairn+memory+architecture&budget=1500` (memories from docs
31-33 are still in the store, so this exercises the real cross-modal ranking, not an
empty-corpus edge case).
**Expected**: `included[]` contains a mix of `"source":"memory"` and `"source":"doc"` items,
budget respected.
**Observed**: `used_tokens: 1490/1500`; positions 0-2 are `source: "memory"` (higher-scored, from
docs 31-33's seeded memories), positions 3-4 are `source: "doc", kind: "doc"` with real README
content - confirms Sprint 6's cross-modal score normalization is blending both sources into one
ranked list, not just appending documents at the end.
**Result**: PASS

### Step 5: `cairn documents delete` removes it
**Do**: `cairn documents delete b33563055168`, then re-check `GET /api/documents`.
**Expected**: prints `Deleted <id>.`; the document list is empty afterward.
**Observed**: `Deleted b33563055168.`; `GET /api/documents` -> `[]`.
**Result**: PASS
